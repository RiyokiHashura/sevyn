## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/RiyokiHashura/sevyn.git
```

2. Install dependencies:
```bash
cd sevyn
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
